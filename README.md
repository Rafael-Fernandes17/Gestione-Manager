# Gestione-Manager

Passo a passo para que o projeto dê certo:

1- Clone o repositorio git, para isso use esse comando no terminal do vsCode:
```bash 
git clone https://github.com/Rafael-Fernandes17/Gestione-Manager.git
```

2- Deixe o seu repositorio o mais atualizado possível todas as vezes que forem codar, para isso, usem esses comandos:

```bash 
git checkout main
```
```bash 
git pull origin main
```

3- Crie a sua branch. 
Para mantermos um projeto organizado, vamos nomear as branchs seguindo esse padrão: primeiro nome da pessoa + "-" + Função que está desenvolvendo. EX:
O nome da minha branch seria: rafael-login.
Para criar e entrar na sua branch, use o comando:
```bash 
git checkout -b nomeDaSuaBranch
```

4- Aqui você coda o que tiver que codar

5- Para criarmos e enviarmos os commits, precisamos fazer:
```bash 
git add .
```

```bash 
git commit -m "mensagem do commit"
```

```bash 
git push origin nomeDaSuaBranch
```

6- Depois é só abrir dentro do gitHub mesmo o pull request
